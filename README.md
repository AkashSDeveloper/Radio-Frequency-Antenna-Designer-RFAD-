# Radio Frequency Antenna Designer (RFAD)

## Overview

A telecom antenna design application developed using C#, JavaScript, JSON, REST APIs, and MS SQL Server. The system enables RF Engineers to create, configure, and manage custom radio frequency antenna designs for telecom networks. It streamlines antenna design processes, improves design accuracy, and centralizes design data management.
